# Problem Statement

Schools and educational institutions still perform many administrative tasks manually such as attendance management, student record maintenance, parent-teacher communication, and PTM scheduling.

Manual systems consume teacher time, increase paperwork, and reduce communication efficiency between teachers and parents.

The proposed system, EduSync AI, is a smart school management web application that automates attendance tracking, student records, parent communication, PTM scheduling, and report generation through a centralized digital platform.

The system aims to reduce teacher workload and improve school management efficiency.
