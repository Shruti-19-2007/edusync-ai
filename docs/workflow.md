# Project Workflow

1. User logs into the system.

2. User dashboard loads according to role:
   - Admin
   - Teacher
   - Parent
   - Student

3. Teacher performs:
   - Attendance marking
   - Homework upload
   - PTM scheduling

4. Parents can:
   - Monitor attendance
   - View reports
   - Book PTM meetings

5. Students can:
   - View homework
   - View attendance
   - Track performance

6. System generates:
   - Reports
   - Notifications
   - Analytics
